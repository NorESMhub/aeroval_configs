# aeroval_configs
A Repository containing configuration files for the aeroval part of pyaerocom. The files here are supposed to be run on sigma2 infrastructure.

Aeroval configuration files are supposed to be below the folder `projects/<project name>`.

## how to run the pyaerocom on sigma2 infrastructure
The repository also contains the receipes to run pyaerocom on the sigma2 infrastructure using `apptainer` as a container solution.
